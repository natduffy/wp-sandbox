This plugin will add a configurable widget to display social media icons in your widget area(s).

== Description ==

This plugin will add a configurable widget to display social media icons in your widget area(s). Icons are 32x32, squared edges, and display inline.

Supported social media links include:

*   Twitter
*   Facebook
*   Google+
*   LinkedIn
*   YouTube
*   Pinterest
*   RSS

== Installation ==

1. Upload 'simple-social-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to your widgets control panel
4. Add the 'Custom - Simple Social Widget' to a widget area and add the title and absolute path(s) to the desired social media sites. (If the input field is left blank, the icon will not appear on the website)
